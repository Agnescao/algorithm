二分法的summary： 
通过每次对比有序数组的一半的值来查询界定要查询的数字在数组的哪一半中，
一开始取整个数组的中间值对比，如果这个数字<整个数组的中间值，那么这个数字就可能存在于数组的左边界里面， 如果不是就可能存在右边
然后我们继续根据上面同样的条件在左一半或者右一半的数组中的中点查找是否这个数字存在， 以此类推，如果在这个过程中找到，我们就返回true,
如果没有一直循环上面的操作直到左边界=右边界

1. sorted array
2. 有左右边界，l,r 记录当前循环的左右点
3. 求得当前左右点的中点m = l+((r-l)>>1)
4. if nums[m]== num return true
5. else if nums[m]> num 这个中点数比 要查的数字大，说明这个要查的数可能在数组的中点的左边部分，那么就将右边界移到中点前一个位置 r= m-1;
6. else nums[m]<num 说明这个数可能在中点往右的部分， 那么这个时候更新子数组的左边界 l = m+1
7. 循环从3-6的步骤直到要么找到了num 或者l=r
